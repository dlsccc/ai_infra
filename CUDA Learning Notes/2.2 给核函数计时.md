# 理论界限最大化
了解各个设备的理论基线，可以更明确知道优化的情况。
```
- Tesla K10 单精度峰值浮点数计算次数：745MHz核心频率 x 2GPU/芯片 x（8个多处理器 x 192个浮点计算单元CUDA core） x 2 OPS/周期 =4.58 TFLOPS  (每秒进行4.58万亿次单精度浮点运算)
- Tesla K10 内存带宽峰值： 2GPU/芯片 x 256 位（每个GPU显存总线宽度） x 2500 MHz内存时钟 x 2 DDR/8位/字节 = 320 GB/s
- 计算/访存比（FLOPs/Byte）：字节 4.58 TFLOPS/320 GB/s =14.3 个浮点计算： 1个字节
```
